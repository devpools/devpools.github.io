    사용법

    아직 빌드중이지만 아래와 같이 설계됨

    설치

    `$npm install
`

    실행

    `$npm start
`

    이렇게 실행하고 http://localhost:3000 으로 접속하시면 개발 버전을 확인할 수 있습니다.

    `npm run deploy

를 통해 작업할 수 있습니다.
